’di ni | | med par byaṅ chub med par gsuṅs | |

gal te theg chen ma grub na | | khyod kyi luṅ ni ji ltar grub | |gaṅ phyir gñis ka la ’di grub
| | daṅ po khyod la ’di ma grub | |

rkyen gaṅ gis ni der yid ches | | de ni theg chen la yaṅ mtshuṅs | |gźan gñis ’dod pas bden
na ni | | rig byed sogs kyaṅ bden par ’gyur | |

dag | | rtsod bcas yin phyir dor byar ’gyur | |

bstan rtsa dge sloṅ ñid yin na | | dge sloṅ ñid kyaṅ dka’ bar gnas | |sems ni dmigs daṅ bcas
rnams kyi | | mya ṅan ’das pa’aṅ dka’ bar gnas | |

ñon moṅs spaṅs pas grol na de’i | | de ma thag tu der ’gyur ro | |ñon moṅs med kyaṅ de dag la
| | las kyi nus pa mthoṅ ba yin | |

re źig ñer len sred pa ni | | med ces ṅes pa ñid ce na | |sred de ñon moṅs can min yaṅ | | kun
rmoṅs bźin du ci ste med | |

tshor ba’i rkyen gyis sred pa yin | | tshor ba de dag la yaṅ yod | |dmigs pa daṅ ni bcas pa’i
sems | | ’ga’ źig la ni gnas par ’gyur | |

stoṅ ñid daṅ ni bral ba’i sems | | ’gags pa slar yaṅ skye ’gyur te | |’du śes med pa’i sñoms
’jug bźin | | des na stoṅ ñid bsgom par bya | |

ṅag gaṅ mdo sde la ’jug de | | gal te saṅs rgyas gsuṅs ’dod na | |theg chen phal cher khyed
cag gi | | mdo daṅ mtshuṅs ’dod min nam ci | |

gal te ma gtogs gcig gis ni | | thams cad skyon daṅ bcas ’gyur na | |mdo mtshuṅs gcig gis thams
cad ni | | rgyal bas gsuṅs pa cis ma yin | |

ṅag gaṅ ’od sruṅs chen po la | | sogs pas gtiṅ dpogs ma gyur pa | |de ni khyod kyis ma rtogs
pas | | gzuṅ bya min par su źig byed | |

rmoṅs pas sdug bsṅal can don du | | chags daṅ ’jigs mtha’ las grol ba | |’khor bar gnas pa grub
’gyur ba | | ’di ni stoṅ ñid ’bras bu yin | |

de ltar stoṅ pa ñid phyogs la | | sun ’byin pa ni ’thad ma yin | |des na the tshom mi za bar
| | stoṅ pa ñid ni bsgom par bya | |

ñon moṅs śes bya’i sgrib pa yi | | mun pa’i gñen po stoṅ pa ñid | |myur du thams cad mkhyen
’dod pas | | de ni ji ltar sgom mi byed | |

| | ’jigs par ’gyur ba su źig yin | |
