gal te sgra yi śes rtag na | | thams cad tshe na sgra ’dzin ’gyur | |śes bya med na ci źig rig
| | gaṅ gis śes pa źes rjod byed | |

gal te śes med śes yin na | | śiṅ yaṅ śes par thal bar ’gyur | |des na śes bya ñer gnas pa |
| med par śes pa med ces ṅes | |

de ñid kyis ni gzugs śes na | | de tshe thos pa’aṅ ci ste min | |gal te sgra mi ñe phyir na
| | des na de yi śes pa’aṅ med | |

sgra ’dzin raṅ bźin gaṅ yin pa | | de gzugs ’dzin par ji lta ’gyur | |gcig ni pha daṅ bu ñid
du | | brtags yin yaṅ dag ñid min te | |

’di ltar sñiṅ stobs rdul daṅ ni | | mun pa bu min pha yaṅ min | |de ni sgra ’dzin daṅ ldan pa’i
| | raṅ bźin du ni mthoṅ ma yin | |

gar mkhan lta bur tshul gźan gyis | | de ñid mthoṅ na de rtag min | |gal te tshul gźan de ñid
na | | gcig ñid de ni sṅon med gcig | |

gal te tshul gźan bden min na | | de yi raṅ gi raṅ bźin smros | |śes ñid ce na de lta na | |
skyes kun gcig tu thal bar ’gyur | |

sems pa sems med de dag kyaṅ | | gcig ’gyur gaṅ phyir yod ñid mtshuṅs | |gaṅ tshe bye brag phyin
ci log | | de tshe ’bra ba’i rten gaṅ yin | |

sems med pa yaṅ bdag min te | | sems med ñid phyir bum sogs bźin | |’on te sems daṅ ldan pa’i
phyir | | śes na mi śes ’jig par thal | |

ji ste bdag la ’gyur med na | | sems pas de la ci źig byas | |de ltar śes med bya bral ba |
| nam mkha’ bdag tu byas par ’gyur | |

gal te bdag ni yod min na | | las ’bras ’brel pa rigs min te | |las byas nas ni źig pas na |
| su yi las su ’gyur źe na | |

bya ba ’bras bźi tha dad daṅ | | de bdag byed pa med par yaṅ | |yu bu gñi ga la grub pas | |
’dir brtsad don med ma yin nam | |

rgyu ldan ’bras bu daṅ bcas źes | | mthoṅ ba ’di ni srid ma yin | |rgyud gcig la ni brten nas
su | | byed po spyod po źes byar bstan | |

’das daṅ ma ’oṅs pa yi sems | | bdag min de ni med pa’i phyir | |’on te skyes sems bdag yin
na | | de źig na yaṅ bdag med do | |

dper na chu śiṅ sdoṅ po dag | | cha śas phye na ’ga’ med pa | |de bźin rnam par dpyad pa yis
| | btsal na bdag kyaṅ yaṅ dag min | |

gal te sems can yod min na | | su la sñiṅ rje bya źe na | |’bras bu’i don du khas blaṅs pa’i
| | rmoṅs pas brtags pa gaṅ yin pa’o | |

sems can med ’bras su yi yin | | bden te ’on kyaṅ rmoṅs las ’dod | |sdug bsṅal ñe bar źi don
du | | ’bras bu’i rmoṅs pa bzlog mi bya | |

sdug bsṅal rgyu yi ṅa rgyal ni | | bdag tu rmoṅs pas ’phel bar ’gyur | |de las kyaṅ bzlog med
ce na | | bdag med bsgom pa mchog yin no | |

te | | braṅ daṅ dpuṅ pa’aṅ lus ma yin | |
