yin | | dṅos ’gyur gźan yaṅ gaṅ źig yin | |

ni | | rtag tu ma skyes ma ’gags ñid | |

| | de ñid du na khyad par med | |

| | yoṅs su brñas pa’aṅ ci źig yod | |

| | gaṅ źig sred ciṅ gaṅ la sred | |
