spyod pa ’di ’dras gnas byed na | | skye bo ṅan pa’i khrod gnas sam | |bud med khrod na gnas
kyaṅ ruṅ | | sdom brtson brtan pa ñams mi ’gyur | |

bdag gi rñed daṅ bkur sti daṅ | | lus daṅ ’tsho ba med bla źiṅ | |dge ba gźan yaṅ ñams bla yi
| | sems ni nams kyaṅ ñams mi bya | |

sems bsruṅ ’dod pa rnams la ni | | dran pa daṅ ni śes bźin dag | |thams cad ’bad pas sruṅs śig
ces | | bdag ni de ltar thal mo sbyor | |

nad kyis dkrugs pa’i mi dag ni | | las rnams kun la mthu med pa | |de bźin rmoṅs pas sems dkrugs
pa | | las rnams kun la mthu med do | |

śes bźin med pa’i sems ldan pa’i | | thos daṅ bsams daṅ bsgoms pa yaṅ | |glo rdol bum pa’i chu
bźin du | | dran pa la ni de mi gnas | |

thos ldan dad pa can daṅ ni | | brtson pa lhur len du ma yaṅ | |śes bźin med pa’i skyon chags
pas | | ltuṅ ba’i rñog daṅ bcas par ’gyur | |

śes bźin med pa’i chom rkun dag | | dran pa ñams pa’i rjes ’braṅ bas | |bsod nams dag ni ñer
bsags kyaṅ | | rkun pos phrogs bźin ṅan ’gror ’gro | |

ñon moṅs chom rkun tshogs ’di ni | | glags skabs tshol bar byed pa ste | |glags rñed gyur nas
dge ’phrog ciṅ | | bde ’gro’i srog kyaṅ ’joms par byed | |

de bas dran pa yid sgo nas | | gud du nam yaṅ mi gtoṅ ṅo | |soṅ na’aṅ ṅan ’gro gnod pa dag |
| dran par byas te ñe bar bźag | |

bla ma daṅ ni ’grogs pa las | | mkhan pos rjes su brtan pa daṅ | |’jigs pas skal ldan gus byed
la | | dran pa bde blag ñid du skye | |

saṅs rgyas byaṅ chub sems dpa’ dag | | kun du thogs med gzigs par ldan | |de dag thams cad spyan
sṅa na | | rtag par bdag ni gnas so źes | |

de ltar bsams nas ṅo tsha daṅ | | gus daṅ ’jigs ldan de bźin gnos | |des na saṅs rgyas rjes
dran pa’aṅ | | de la yaṅ daṅ yaṅ du ’byuṅ | |

gaṅ tshe dran pa yid sgo nas | | bsruṅ pa’i don du gnas gyur pa | |de tshe śes bźin ’oṅ ’gyur
źiṅ | | soṅ ba dag kyaṅ ’oṅ bar ’gyur | |

re źig daṅ po ’di ’dra’i sems | | ’di ni skyon bcas śes byas nas | |de tshe bdag gis śiṅ bźin
du | | zuṅs thub par ni gnas par bya | |

don med g-yeṅ bar lta ba ni | | nam yaṅ bdag gis mi bya ste | |ṅes par sems pas rtag tu ni |
| mig ni phab ste blta bar bya | |

lta ba ṅal bso’i ched du ni | | res ’ga’ phyogs su blta bar bya | |’ga’ źig mig lam snaṅ gyur
na | | bltas nas ’oṅs pa legs źes brjod | |

bltas nas | | rgyab kyi phyogs su blta bar bya | |

cad du | | dgos pa śes nas spyad par bya | |

| | gnas pa yin źes blta bar bya | |
