rgyud daṅ tshogs źes bya ba ni | | phreṅ ba dmag la sogs bźin brdzun | |sdug bsṅal can gaṅ de
med pa | | des ’di su źig spaṅ bar ’gyur | |

sdug bsṅal bdag po med par ni | | thams cad bye brag med pa ñid | |sdug bsṅal yin phyir de bsal
bya | | ṅes pas der ni ci źig bya | |

ci phyir kun gyi sdug bsṅal ni | | bzlog par bya źes brtsad du med | |gal te bzlog na’aṅ thams
cad bzlog | | de min bdag kyaṅ sems can bźin | |

sñiṅ rje sdug bsṅal maṅ gyur pa | | ci phyir nan gyis skyed ce na | |’gro ba’i sdug bsṅal bsam
byas na | | ji ltar sñiṅ rje sdug bsṅal maṅ | |

gal te sdug bsṅal gcig gis ni | | sdug bsṅal maṅ po med ’gyur na | |brtse daṅ ldan pas sdug
bsṅal de | | raṅ daṅ gźan la bskyed bya ñid | |

des na me tog zla mdzes kyis | | rgyal po’i gnod pa śes kyaṅ ni | |bdag gi sdug bsṅal ma bsal
te | | maṅ po’i sdug bsṅal zad ’gyur phyir | |

de ltar rgyud ni goms gyur pa | | gźan gyi sdug bsṅal źi dga’ bas | |pa dma’i mtsho ru ṅaṅ pa
ltar | | mnar med pa yaṅ ’jug par ’gyur | |

sems can rnam par grol ba na | | dga’ ba’i rgya mtsho gaṅ yin pa | |de ñid kyis ni chog min
nam | | thar pa ’dod pas ci źig bya | |

de ltas gźan gyi don byas kyaṅ | | rlom sems daṅ ni ṅo mtshar med | |gcig tu gźan don la dga’
bas | | rnam smin ’bras bu’i re ba med | |

de bas ji ltar chuṅ ṅu na | | mi sñan las kyaṅ bdag bsruṅ ba | |de bźin gźan la bsruṅ sems daṅ
| | sñiṅ rje’i sems ni de ltar bya | |

goms pa yis ni gźan dag gi | | khu ba khrag gi thigs pa la | |dṅos po med par gyur kyaṅ ni |
| bdag go źes ni śes pa ltar | |

de bźin gźan gyi lus la yaṅ | | bdag ces ci yi phyir mi gzuṅ | |bdag gi lus ni gźan dag tu’aṅ
| | bźag pa de ltar dka’ ba med | |

bdag ñid skyon bcas gźan la yaṅ | | yon tan rgya mtshor śes byas nas | |bdag ’dzin yoṅs su dor
ba daṅ | | gźan blaṅ ba ni bsgom par bya | |

ji ltar lag pa la sogs pa | | lus kyi yan lag yin ’dod ltar | |de bźin ’gro ba’i yan lag tu
| | ci phyir lus can rnams mi ’dod | |

ji ltar bdag med lus ’di la | | goms pas bdag gi blo byuṅ ba | |de bźin sems can gźan la yaṅ
| | goms pas bdag blo cis mi skye | |

de lta na ni gźan gyi don | | byas kyaṅ ṅo mtshar rlom mi ’byuṅ | |bdag ñid kyis ni zas zos
nas | | lan la re ba mi ’byuṅ bźin | |

de bas ji ltar chuṅ ṅu na | | mi sñan las kyaṅ bdag bsruṅ ba | |de bźin ’gro la bsruṅ sems daṅ
| | sñiṅ rje’i sems ni goms par bya | |

de bas mgon po spyan ras gzigs | | thugs rje chen pos ’gro ba yi | |’khor gyi ’jigs pa bsal
ba’i phyir | | raṅ gi mtshan yaṅ byin gyis brlabs | |

dka’ las phyir ldog mi bya ste | | ’di ltar goms pa’i mthu yis ni | |gaṅ gi miṅ thos ’jigs pa
yaṅ | | de ñid med na mi dgar ’gyur | |

gaṅ źig bdag daṅ gźan rnams ni | | myur du bskyab par ’dod pa des | |bdag daṅ gźan du brje bya
ba | | gsaṅ ba’i dam pa spyad par bya | |
