tsam mthoṅ nas | | ci phyir de la gduṅ bar byed | |

gso bya gaṅ źig khyod gso ba | | khyod ñid kyis ni sbyin bya de | |gñen gyis ’tsho ba rñed gyur
na | | dgar mi ’gyur bar slar khro’am | |

de ci ’gro la mi ’dod na | | de la byaṅ chub ’dod gaṅ yin | |gaṅ źig gźan ’byor khro de la |
| byaṅ chub sems ni ga la yod | |

gal te des de las rñed dam | | sbyin bdag khyim na gnas gyur pa | |kun du de khyod la med na
| | byin ma byin ruṅ des ci bya | |

bsod nams daṅ ni dad pa’am | | raṅ gi yon tan ci phyir ’dor | |rñed pa gyur pa mi ’dzin pa |
| gaṅ gis khro ba mi byed smros | |

khyod ni raṅ gis sdig byas la | | mya ṅan med par ma zad kyi | |bsod nams byas pa gźan dag daṅ
| | lhan cig ’gran par byed ’dod dam ||

|| bam po gñis pa’o | |

gal te dgra źig mi dga’ na’aṅ | | de la khyod dgar ci źig yod | |khyod kyi yid smon tsam gyis
ni | | de la gnod pa’i rgyur mi ’gyur | |

khyod kyi ’dod pas sdug bsṅal de | | grub na’aṅ khyod dgar ci źig yod | |gal te ’tshaṅs par
’gyur źe na | | de las phuṅ ba’aṅ gźan ci yod | |

ñon moṅs ña pas btab pa yi | | mchil pa ’di ni mi bzad gze | |des bzuṅ sems dmyal bum par yaṅ
| | dmyal ba’i sruṅ mas bdag ’tshed des | |

bstod daṅ grags pa’i rim gros ni | | bsod nams mi ’gyur cher mi ’gyur | |bdag stobs mi ’gyur
nad med min | | lus bde bar yaṅ mi ’gyur ro | |

bdag ni raṅ don śes gyur na | | de la raṅ don ci źig yod | |yid bde ’ba’ źig ’dod na ni | |
rgyan sogs chad yaṅ bsten dgos so | |

grags pa’i don du nor śor ciṅ | | bdag ñid kyaṅ ni gsod byed na | |tshig ’bru rnams kyis ci
źig bya | | śa ni des ko su la bde | |

phye ma’i khaṅ bu rdib gyur na | | byis pa rnams ni ga chad du | |de bźin bstod daṅ grags ñams
na | | raṅ gi sems ni byis pa bźin | |

re źig sgra la sems med phyir | | bdag la bstod sems yod mi srid | |bdag la gźan dga’ źes grags
pa | | de ni dga’ ba’i rgyu yin graṅ | |

gźan nam yaṅ na bdag la’aṅ ruṅ | | gźan dga’ bdag la ci źig phan | |dga’ bde de ni de ñid kyi
| | bdag gis de las śas mi ’thob | |

de bde ba yi bdag bde na | | kun la’aṅ de bźin bya dgos na | |ji ltar gźan la dga’ ba yis |
| bde bar gyur na bdag mi bde | |

de bas bdag ni bstod do źes | | raṅ gi dga’ ba skye ’gyur ba | |de yaṅ de ltar mi ’thad pas
| | byis pa’i spyod pa kho nar zad | |

dog daṅ | | phun sum tshogs pa’aṅ ’jig par byed | |

du | | ltuṅ ba bsruṅ phyir źugs min nam | |
