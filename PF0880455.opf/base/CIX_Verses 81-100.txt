mod | | de raṅ ñid ni gaṅ la gnas | |

gal te bdag ñid kun gyi lus | | lag sogs rnams la gnas na ni | |lag sogs de dag ji sñed pa |
| de sñed kyi ni lus su ’gyur | |

phyi daṅ naṅ na lus med na | | ji ltar lag sogs la lus yod | |lag sogs rnams las gźan med na
| | de ni ji ltar yod pa yin | |

des na lus med la sogs la | | rmoṅs pa yis ni lus blor ’gyur | |dbyibs su bkod pa’i khyad par
gyis | | tho yor la ni mi blo bźin | |

ji srid rkyen tshogs de srid du | | lus ni skyes bu ltar snaṅ ba | |de bźin ji srid lag sogs
la | | de yod de srid de lus snaṅ | |

de bźin sor mo’i tshogs yin phyir | | lag pa’aṅ gaṅ źig yin par ’gyur | |de yaṅ tshigs kyi tshogs
yin phyir | | tshigs kyaṅ raṅ gi cha phye bas | |

cha yaṅ rdul du phye bas te | | rdul de’aṅ phyogs cha’i dbye ba yis | |phyogs dbye’aṅ cha śas
daṅ bral phyir | | mkha’ bźin des na rdul yaṅ med | |

de ltar rmi lam lta bu yi | | gzugs la dpyod ldan su źig chags | |gaṅ tshe de ltar lus med pa
| | de tshe skyes gaṅ bud med gaṅ | |

sdug bsṅal de ñid du yod na | | ci ste rab dga’ la mi gnod | |bde na mya ṅan gduṅ sogs la |
| źim sogs ci ste dga’ mi byed | |

stobs daṅ ldan pas zil mnan phyir | | gal te de myoṅ ma yin na | |gaṅ źig ñams myoṅ bdag ñid
min | | de ni tshor ba ji ltar yin | |

sdug bsṅal phra mo ñid du yod | | ’di yi rags pa bsal min nam | |de ni de las gźan dga’ tsam
| | źe na phra ñid de yaṅ de’i | |

gal te ’gal rkyen skyes pas na | | sdug bsṅal skyes pa min na ni | |tshor bar rtog pa mṅon źen
ñid | | yin źes bya bar grub min nam | |

de ñid phyir na ’di yi ni | | gñen po rnam dpyod ’di bsgom ste | |rnam brtags źiṅ las byuṅ ba
yi | | bsam gtan rnal ’byor pa yi zas | |

gal te dbaṅ don par bcas na | | de dag gaṅ du phrad par ’gyur | |bar med na yaṅ gcig ñid de
| | gaṅ źig gaṅ daṅ phrad par ’gyur | |

rdul phran rdul phran la ’jug med | | de ni skabs med mñam pa yin | |ma źugs pa la ’dre med
ciṅ | | ma ’dres pa la phrad pa med | |

cha med pa la’aṅ phrad pa źes | | bya ba ji ltar ’thad par ’gyur | |phrad pa daṅ ni cha med
par | | gal te mthoṅ na bstan par gyis | |

rnam śes lus med pa la ni | | phrad pa ’thad pa ma yin ñid | |tshogs pa’aṅ dṅos po med phyir
te | | sṅar ni ji ltar rnam dpyad bźin | |

de ltar reg pa yod min na | | tshor ba gaṅ las ’byuṅ bar ’gyur | |ṅal ’di ci yi don du yin |
| gaṅ gis gaṅ la gnod par ’gyur | |

nas | | sred pa ci phyir ldog mi ’gyur | |
