rgyal ba’i sras kyis de lta bur | | byaṅ chub sems rab brtan bzuṅ nas | |g-yel ba med par rtag
tu yaṅ | | bslab las mi ’da’ ’bad par bya | |

bab col brtsam pa gaṅ yin pa’am | | gaṅ źig legs par ma brtags pa | |de ni dam bcas byas gyur
kyaṅ | | bya’am btaṅ źes brtags pa’i rigs | |

saṅs rgyas rnams daṅ de yi sras | | śes rab chen pos gaṅ brtags śiṅ | |bdag ñid kyis kyaṅ brtag
brtags pa | | de la bśol du ci źig yod | |

gal te de ltar dam bcas nas | | las kyis bsgrub pa ma byas na | |sems can de dag kun bslus pas
| | bdag gi ’gro ba ci ’drar ’gyur | |

dṅos po phal pa cuṅ zad la’aṅ | | yid kyis sbyin par bsam byas nas | |mi gaṅ sbyin par mi byed
pa | | de yaṅ yi dags ’gyur gsuṅs na | |

bla na med pa’i bde ba la | | bsam pa thag pas mgron gñer nas | |’gro ba thams cad bslu byas
na | | bde ’gror ji ga ’gro ’gyur ram | |

mi gaṅ byaṅ chub sems btaṅ yaṅ | | de dag thar par byed pa ni | |las tshul bsam gyis mi khyab
ste | | thams cad mkhyen pa kho nas mkhyen | |

de ni byaṅ chub sems dpa’ la | | ltuṅ ba’i naṅ na lci ba ste | |’di ltar de ni byuṅ gyur na
| | sems can kun gyi don la dman | |

gaṅ gźan skad cig tsam yaṅ ’di’i | | bsod nams bar chad gegs byed pa | |sems can don la dman
gyur pas | | de yi ṅan ’gro mu mtha’ med | |

sems can gcig gi bde ba yaṅ | | bśig na bdag ñid ñams ’gyur na | |nam mkha’ ma lus mtha’ klas
pa’i | | lus can bde bśig smos ci dgos | |

de ltar ltuṅ ba stobs ldan daṅ | | byaṅ chub sems stobs ldan pa dag | |’khor bar res kyis ’dre
byed na | | sa thob pa la yun riṅ thogs | |

de ltas ji ltar dam bcas bźin | | bdag gis gus par bsgrub par bya | |deṅ nas brtson par ma byas
na | | ’og nas ’og tu ’gro bar ’gyur | |

sems can thams cad phan mdzad pa’i | | saṅs rgyas graṅs med ’das gyur kyaṅ | |bdag ni raṅ gi
ñes pas de’i | | gso ba’i spyod yul ma gyur to | |

da duṅ de bźin bdag byed na | | yaṅ daṅ yaṅ du’aṅ de bźin te | |ṅan ’gror nad daṅ ’chiṅ ba daṅ
| | bcad daṅ gśeg sogs myoṅ bar ’gyur | |

de bźin gśegs pa ’byuṅ ba daṅ | | dad daṅ mi lus thob pa daṅ | |dge goms ruṅ ba de lta bu |
| dkon pa nam źig thob par ’gyur | |

nad med ñi ma ’di lta bu | | zas bcas ’tshe ba med kyaṅ ni | |tshe ni skad cig bslu ba ste |
| lus ni thaṅ cig brñan po bźin | |

bdag gi spyod pa ’di ’dras ni | | mi yi lus kyaṅ ’thob mi ’gyur | |mi lus thob par ma gyur na
| | sdig pa ’ba’ źig dge ba med | |

gaṅ tshe dge spyad skal ldan yaṅ | | dge ba bdag gis ma byas na | |ṅan soṅ sdug bsṅal kun rmoṅs
pa | | de tshe bdag gis ci byar yod | |

dge ba dag kyaṅ ma byas la | | sdig pa dag kyaṅ ñer bsags na | |bskal pa bye ba brgyar yaṅ ni
| | bde ’gro’i sgra yaṅ thos mi ’gyur | |

chud ltar | | mi ñid śin tu thob dkar gsuṅs | |
