gźan yaṅ sdug bsṅal yon tan ni | | skyo bas dregs pa sel bar byed | |’khor ba pa la sñiṅ rje
ste | | sdig la ’dzem daṅ dge la dga’ | |

mkhris pa la sogs sdug bsṅal gyi | | ’byuṅ gnas che la mi khro bar | |sems yod rnams la ci ste
khro | | de dag kun kyaṅ rkyen gyis bskul | |

dper na mi ’dod bźin du yaṅ | | nad ’di ’byuṅ bar ’gyur ba ltar | |de bźin mi ’dod bźin du yaṅ
| | nan gyis ñon moṅs ’byuṅ bar ’gyur | |

khro bar bya źes ma bsams kyaṅ | | skye bo rnams ni gyi nar khro | |bskyed par bya źes ma bsams
kyaṅ | | khro ba de bźin skye bar ’gyur | |

ñes pa ji sñed thams cad daṅ | | sdig pa rnam pa sna tshogs pa | |de kun rkyen gyi stobs las
byuṅ | | raṅ dbaṅ yod pa ma yin no | |

rkyen rnams tshogs pa de dag kyaṅ | | bskyed par bya źes sems pa med | |des bskyed de yaṅ bdag
bskyed cis | | sems pa yod pa ma yin no | |

gtso bo źes byar gaṅ ’dod daṅ | | bdag ces brtags pa gaṅ yin pa | |de ñid bdag ni ’byuṅ bya
źes | | ched du bsams śiṅ ’byuṅ ba med | |

ma skyes par ni de med na | | de tshe skye bar ’dod pa gaṅ | |yul la rtag tu g-yeṅ ’gyur ba
| | ’gag par ’gyur ba’aṅ ma yin no | |

ci ste bdag de rtag na ni | | mkha’ bźin byed pa med par mṅon | |rkyen rnams gźan daṅ phrad
na yaṅ | | ’gyur ba med la ci byar yod | |

byed pa’i tshe yaṅ sṅon bźin na | | byed pas de la ci źig byas | |de yi byed pa ’di yin źes
| | ’brel par ’gyur ba gaṅ źig yod | |

de ltar thams cad gźan gyi dbaṅ | | de yi dbaṅ gis de dbaṅ med | |de ltar śes na sprul lta bu’i
| | dṅos po kun la khro mi ’gyur | |

gaṅ gis gaṅ źig bzlog bya ste | | bzlog pa’aṅ rigs pa min źe na | |de la brten nas sdug bsṅal
rnams | | rgyun chad ’gyur ’dod mi rigs med | |

de bas dgra’am mdza’ yaṅ ruṅ | | mi rigs byed pa mthoṅ gyur na | |’di ’dra’i rkyen las gyur
to źes | | de ltar soms te bde bar mnos | |

gal te raṅ dags ’grub ’gyur na | | ’ga’ yaṅ sdug bsṅal mi ’dod pas | |lus can dag ni thams cad
kyaṅ | | su la’aṅ sdug bsṅal ’byuṅ mi ’gyur | |

bag med pas ni bdag la’aṅ bdag | | tsher ma la sogs gnod pa byed | |bud med la sogs thob bya’i
phyir | | rṅam źiṅ zas gcod la sogs byed | |

kha cig ’gag ciṅ g-yaṅ sar mchoṅ | | dug daṅ mi ’phrod za ba daṅ | |bsod nams ma yin spyod pa
yis | | raṅ la gnod pa byed pa yod | |

gaṅ tshe ñon moṅs dbaṅ gyur pas | | bdag sdug ñid kyaṅ gsod byed pa | |de tshe de dag gźan lus
la | | gnod mi byed par ji ltar ’gyur | |

ñon moṅs skyes pas de lta bur | | bdag gsod la sogs źugs pa la | |sñiṅ rje rgya la ma skyes
na | | khro bar ’gyur ba ci tha tshig | |

gal te gźan la ’tshe byed pa | | byis pa rnams kyi raṅ bźin ni | |de la khro ba mi rigs te |
| sreg pa’i raṅ bźin me bkon ’dra | |

’on te skyon de glo bur la | | sems can raṅ bźin des pa na’aṅ | |’o na’aṅ khro ba mi rigs te
| | mkha’ la dud ’thul dkon pa bźin | |
