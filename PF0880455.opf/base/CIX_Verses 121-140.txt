dbaṅ phyug mkha’ min g-yo med phyir | | bdag min sṅar ni bkag zin phyir | |bsam min pa yi byed
po yaṅ | | bsam min brjod pas ci źig bya | |

des bskyed ’dod pa’aṅ gaṅ źig yin | | bdag ni de daṅ sa sogs daṅ | |dbaṅ phyug ṅo bo’aṅ rtag
min nam | | śes pa śes bya las skye daṅ | |

thog med bde sdug las las yin | | de yis gaṅ źig bskyed pa smros | |rgyu la thog ma yod min
na | | ’bras bu’i thog ma ga la yod | |

rtag tu ci phyir byed min te | | de ni gźan la ltos pa min | |des byas min gźan yod min na |
| des ’di gaṅ la ltos par ’gyur | |

gal te ltos na tshogs pa ñid | | rgyu yin ’gyur gyi dbaṅ phyug min | |tshogs na mi skye dbaṅ
med ciṅ | | de med par ni skye dbaṅ med | |

gal te dbaṅ phyug mi ’dod bźin | | byed na gźan gyi dbaṅ du thal | |’dod na’aṅ ’dod la rag las
’gyur | | byed na’aṅ dbaṅ phyug ga la yin | |

gaṅ dag rdul phran rtag smra ba | | de dag kyaṅ ni sṅar bzlog zin | |gtso bo rtag pa ’gro ba
yi | | rgyu yin par ni graṅs can ’dod | |

sñiṅ stobs rdul daṅ mun pa źes | | bya ba’i yon tan mñam gnas ni | |gtso bo źes byar rab brjod
de | | mi mñam ’gro ba yin par brjod | |

gcig la raṅ bźin gsum ñid ni | | rigs min des na de yod min | |de bźin yon tan yod min te |
| de yaṅ so sor rnam gsum phyir | |

yon tan med na sgra sogs kyaṅ | | yod ñid śin tu rgyaṅ riṅ ’gyur | |sems med gos la sogs pa
la | | bde sogs yod pa srid pa’aṅ min | |

dṅos rnams de rgyu’i raṅ bźin na | | dṅos po rnam dpyad ma zin nam | |khyod kyi rgyu yaṅ bde
sogs ñid | | de las snam sogs ’byuṅ ba’aṅ med | |

snam sogs las ni bde sogs yin | | de med phyir na bde sogs med | |bde sogs rtag pa ñid du yaṅ
| | nam yaṅ dmigs pa yod ma yin | |

bde sogs gsal ba yod ñid na | | myoṅ ba ci phyir ’dzin ma yin | |de ñid phra mor gyur na de
| | rags daṅ phra ba’aṅ ji ltar yin | |

rags pa dor nas phra gyur pas | | phra rags de dag mi rtag ñid | |de bźin dṅos po thams cad
ni | | mi rtag ñid du cis mi ’dod | |

rags pa bde las gźan min na | | bde ba gsal ba mi rtag ñid | |gal te med pa ’ga’ yaṅ ni | |
skye min med phyir źes ’dod na | |

gsal ba med pa skye bar ni | | khyod mi ’dod kyaṅ gnas pa ñid | |gal te rgyu la ’bras gnas na
| | zan za mi gtsaṅ za bar ’gyur | |

ras kyi rin gyis ras bal gyi | | sa bon ños la bgo bar gyis | |’jig rten rmoṅs pas ma mthoṅ
na | | de ñid śes kyis bźag de ñid | |

śes de ’jig rten la yaṅ ni | | yod pas ci ste mthoṅ ma yin | |’jig rten tshad ma ñid min na
| | gsal ba mthoṅ ba’aṅ bden ma yin | |

gal te tshad ma tshad min na | | des gźal brdzun par mi ’gyur ram | |de ñid du na stoṅ pa ñid
| | sgom pa de phyir mi ’thad ’gyur | |

brtags pa’i dṅos la ma reg par | | de yi dṅos med ’dzin ma yin | |de phyir brdzun pa’i dṅos
gaṅ yin | | de yi dṅos med gsal bar brdzun | |
