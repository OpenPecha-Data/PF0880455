gaṅ gi don du bdag gson pa | | de ñid gal te ñams gyur na | |sdig pa ’ba’ źig byed pa yi | |
gson pa des ko ci źig bya | |

gal te sems can ñams byed pas | | mi sñan smra la khro źe na | |gźan la mi sñan brjod la’aṅ
khyod | | de bźin ci ste khro mi byed | |

ma dad gźan la rag las pas | | dad pa med la khyod bzod na | |ñon moṅs skye la rag las pas |
| mi sñan smra la cis mi bzod | |

sku gzugs mchod rten dam chos la | | ’khyal źiṅ ’jig par byed pa la’aṅ | |bdag gis źe sdaṅ mi
rigs te | | saṅs rgyas sogs la gnod mi mṅa’ | |

bla ma gñen la sogs pa daṅ | | bśes la gnod pa byed rnams la’aṅ | |sṅa ma’i tshul gyis rkyen
dag las | | ’gyur bar mthoṅ nas khro ba bzlog | |

lus can rnams la sems yod daṅ | | sems med gñi gas gnod byas na | |sems yod ci ste bkar te bkon
| | de bas gnod pa bzod par gyis | |

la la rmoṅs pas ñes pa byed | | la la rmoṅs te khros gyur na | |de la skyon med gaṅ gis bya
| | skyon daṅ bcas te gaṅ źig yin | |

gaṅ gis gźan dag gnod byed pa’i | | las de sṅon chad ci phyir byas | |thams cad las la rag las
na | | bdag gis ’di la ci ste bkon | |

de ltar mthoṅ nas ce nas kyaṅ | | thams cad phan tshun byams sems su | |’gyur ba de ltar bdag
gis ni | | bsod nams dag la bsgrims te bya | |

dper na khyim tshig gyur pa’i me | | khyim gźan źig tu soṅ nas ni | |rtswa sogs gaṅ la mched
byed pa | | de ni phyuṅ ste dor ba yin | |

de bźin gaṅ la sems chags na | | źe sdaṅ me ni mched gyur pa | |bsod nams tshig par dogs pa
yis | | de ni de yi mod la dor | |

gsad bya’i mi źig lag bcad de | | gal te thar na cis ma legs | |gal te mi yi sdug bsṅal gyis
| | dmyal ba bral na cis ma legs | |

da lta’i sdug bsṅal ’di tsam la’aṅ | | bdag gis bzod par mi nus na | |des na dmyal ba’i sdug
bsṅal rgyu | | khro ba ci ste bzlog mi byed | |

’dod pa’i don du sreg la sogs | | dmyal bar stoṅ phrag myoṅ gyur yaṅ | |bdag gis raṅ gi don
daṅ ni | | gźan gyi don yaṅ ma byas so | |

’di ni de tsam gnod min la | | don chen dag kyaṅ ’grub ’gyur bas | |’gro ba’i gnod sel sdug
bsṅal la | | dga’ ba ’ba’ źig ’dir bya rigs | |

gźan gyis yon tan ldan bstod nas | | gal te dga’ ba’i bde thob na | |yid khyod kyaṅ ni de bstod
nas | | ci phyir de ltar dga’ mi byed | |

khyod kyi dga’ ba’i bde ba ’di | | bde ’byuṅ kha na ma tho med | |yon tan ldan pa rnams kyis
gnaṅ | | gźan sdud pa yi mchog kyaṅ yin | |

gźan yaṅ de ltar bde ’gyur źes | | gal te khyod bde ’di mi ’dod | |gla rṅan sbyin sogs spaṅs
pa’i phyir | | mthoṅ daṅ ma mthoṅ ñams par ’gyur | |

raṅ gi yon tan brjod pa’i tshe | | gźan la bde ba’aṅ ’dod par byed | |gźan gyi yon tan brjod
pa’i tshe | | raṅ la’aṅ bde ba mi ’dod byed | |

sems can thams cad bde ’dod pas | | byaṅ chub tu ni sems bskyed nas | |sems can raṅ gis bde
rñed na | | des ko ci ste khro bar byed | |
