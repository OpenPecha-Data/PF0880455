bsal phyir | | lus rma btod pa’i sdug bsṅal bźin | |

| | gźom phyir mi bde chuṅ bzod bya | |

gso dpyad phal pa ’di ’dra ba | | sman pa mchog gis ma mdzad de | |cho ga śin tu ’jam po yis
| | nad chen dpag med gso bar mdzad | |

nas ni | | rim gyis raṅ gi śa yaṅ gtoṅ | |

gaṅ tshe raṅ gi lus la ni | | tshod sogs lta bu’i blo skyes pa | |de tshe śa la sogs gtoṅ ba
| | de la dka’ ba ci źig yod | |

pa daṅ | | sdig pas sems daṅ lus la gnod | |

bsod nams kyis ni lus bde la | | mkhas pa yis ni sems bde na | |gźan don ’khor bar gnas kyaṅ
ni | | sñiṅ rje can dag ci ste skyo | |

’di ni byaṅ chub sems stobs kyis | | sṅon gyi sdig pa zad byed ciṅ | |bsod nams rgya mtsho sdud
byed phyir | | ñan thos rnams pas mchog tu bśad | |

de bas skyo ṅal kun sel ba’i | | byaṅ chub sems kyi rta źon nas | |bde nas bde bar ’gro ba la
| | sems śes su źig sgyid lug ’gyur | |

pa daṅ | | de yi phan yon bsam pas bskyed | |

kyis ni | | brtson ’grus spel phyir ’bad par bya | |

bdag daṅ gźan gyi ñes pa ni | | dpag med bdag gis gźom bya ste | |gaṅ du ñes pa re re la’aṅ
| | bskal pa rgya mtsho zad ’gyur ba | |

ñes zad rtsom pa de yi ni | | cha yaṅ bdag la ma mthoṅ na | |sdug bsṅal dpag tu med pa’i gnas
| | bdag go ci phyir sñiṅ ma gas | |

| bskal pa rgya mtshos goms ’gyur na | |

| bdag gis don med byas pa mtshar | |
