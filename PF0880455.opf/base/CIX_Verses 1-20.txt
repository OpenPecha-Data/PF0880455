dag | | źi bar ’dod pas śes rab bskyed | |

kun rdzob daṅ ni don dam ste | | ’di ni bden pa gñis su ’dod | |don dam blo yi spyod yul min
| | blo ni kun rdzob yin par brjod | |

de la ’jig rten rnam gñis mthoṅ | | rnal ’byor pa daṅ phal pa ’o | |de la ’jig rten phal pa
ni | | rnal ’byor ’jig rten gyis gnod ciṅ | |

rnal ’byor pa yaṅ blo khyad kyis | | goṅ ma goṅ ma rnams kyis gnod | |gñi ga yaṅ ni ’dod pa’i
dpes | | ’bras bu’i don du ma dpyad phyir | |

’jig rten pa yis dṅos mthoṅ źiṅ | | yaṅ dag ñid du’aṅ rtog byed kyi | |sgyu ma lta bur min pas
’dir | | rnal ’byor pa daṅ ’jig rten rtsod | |

gzugs sogs mṅon sum ñid kyaṅ ni | | grags pas yin gyi tshad mas min | |de ni mi gtsaṅ la sogs
la | | gtsaṅ sogs grags pa bźin du brdzun | |

’jig rten gźug pa’i don du ni | | mgon pos dṅos bstan de ñid du | |de dag skad cig ma ñid min
| | kun rdzob tu yaṅ ’gal źe na | |

rnal ’byor kun rdzob ñes med de | | ’jig rten la ltos de ñid mthoṅ | |gźan du bud med mi gtsaṅ
bar | | des rtog ’jig rten gyis gnod ’gyur | |

sgyu ’dra’i rgyal las bsod nams ni | | ji ltar dṅos yod la ji bźin | |gal te sems can sgyu ’dra
na | | śi nas ji ltar skye źe na | |

ji srid rkyen rnams ’tshogs gyur pa | | de srid sgyu ma’aṅ ’byuṅ bar ’gyur | |rgyun riṅ tsam
gyis ji ltar na | | sems can bden par yod pa yin | |

sgyu ma’i skyes bu bsad sogs la | | sems med phyir na sdig med de | |sgyu ma’i sems daṅ ldan
pa la | | bsod nams daṅ ni sdig pa ’byuṅ | |

sṅags sogs rnams la nus med phyir | | sgyu ma’i sems ni ’byuṅ ba med | |sna tshogs rkyen las
byuṅ ba yi | | sgyu ma de yaṅ sna tshogs ñid | |

rkyen gcig gis ni kun nus pa | | gaṅ na yaṅ ni yod ma yin | |gal te don dam mya ṅan ’das | |
’khor ba kun rdzob de lta na | |

saṅs rgyas kyaṅ ni ’khor ’gyur bas | | byaṅ chub spyod pas ci źig bya | |rkyen rnams rgyun ni
ma chad na | | sgyu ma’aṅ ldog par mi ’gyur gyi | |

rkyen rnams rgyun ni chad pas na | | kun rdzob tu yaṅ mi ’byuṅ ṅo | |gaṅ tshe ’khrul pa’aṅ yod
min na | | sgyu ma gaṅ gis dmigs par ’gyur | |

gaṅ tshe khyod la sgyu ma ñid | | med na’aṅ de tshe ci źig dmigs | |gal te de ñid du gźan yod
| | rnam pa de ni sems ñid yin | |

kyaṅ | | sems kyis sems mi mthoṅ źes gsuṅs | |
